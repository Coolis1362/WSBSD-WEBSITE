Getting Started with WSBSD

Welcome to the WSBSD documentation! Here you'll find all the information you need to get started with WSBSD.

Installation Guide

Follow these steps to install WSBSD:

  1. Download the latest WSBSD build from the Downloads page.
  2. Follow the MSI to complete the installation.
  3. After installation, you can access WSBSD from your applications menu.
  4. For a live terminal experience, visit the Live Terminal page. (IT'S ONLY OPENBSD 7.7)
